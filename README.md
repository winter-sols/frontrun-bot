# SETUP
Install latest Node.js
At the root of the project folder
 ```
$ npm install web3
$ npm install axios
$ npm install colors
$ npm install ethereumjs-tx
$ npm install abi-decoder
 ```
Usage
Enter your PRIVATE_KEY before running the program.
 ```
$ node frontrun.js
 ```
